About this Site


  • This application was developed as part of a 4th year project in college.
  • The developer wanted to create a RESTful API service using the MEAN framework.
    • NodeJs to build the server.
    • Express to develop the routes to the resource in the API.
    • Data on the video game object is stored in JSON format using MongoDB.
    • Angular components were used to build a front end to data in the RESTful API service that stores video game objects.
  • Each video game object in the database contains properties on the video games compatability in an online environment.
  • The listing of video games is transformed when a checkbox button is pressed, or search string is entered to reflect the games compatible with the specified consoles.
